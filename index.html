* Functional programming is fun * Let's have some fun with functional programming In the workshop we will practice functional programming skills in a form of coding dojo. Coding dojo is fun, you know that? We know. And functional programming is fun as well. Our aim is to make this workshop as much of a fun as possible. We will pair and solve some interesting problem using a functional approach in whatever language you want, although we can support you in Scala and Clojure. Each example will be solved during the workshop, while some interesting aspects and solutions will be showed and discussed in the group.
